Toglr

Toggle visibilty of grouped elements.

$( control ).toglr( group, closeAll, addBody, direction, full, actionType, push );

Grouping

Items can be grouped based on a specific container or the full page.

Controls

Each item works with a link that targets the matching container within the group. Each link, Expand, Collapse, Next and Previous are controls that can be used based on the type of component being created.

Navigation Container: 
data-nav="[group-control-name]"
Group Container: 
data-control="[group-control-name]"
