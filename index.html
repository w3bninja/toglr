Toglr

Toglr is a script that can be used to show and hide elements and groups of elements.

Grouping

Items can be grouped based on a specific container or the full page.

Controls

Each item works with a link that targets the matching container within the group. Each link, Expand, Collapse, Next and Previous are controls that can be used based on the type of component being created.

  1. group - The selector of each group.
  2. closeAll - Allow all containers to be closed.
  3. addBody - Add active class to the body.
  4. direction - Add a direction class of left, right, top or bottom.
  5. full - Add a "full" class.
  6. actionType - Adjust the action function trigger.
  7. push - Add "push" classes based on
